Template is based on the custom 960 grid system. It's very easy to split content for parts. If You want to create a content with 1/2 and 1/2 You simply create two div with classes 'one_half' and to the last one add another class 'last_column':

<div class="one_half">

	your content here

</div>
<div class="one_half last_column">

	your content here

</div>

After this You should add 

<div class="divider"></div>
        
	if You want a divider after this div or
        
<div class="clear"></div>

	if You don't or it's just the end of content.

Most common classes You can use for managing Your theme:

  • <div class="one_half"></div>
  • <div class="one_third"></div>
  • <div class="one_fourth"></div>
  • <div class="two_third"></div>
  • <div class="three_fourth"></div>
  • <div class="full_width"></div>

As You can see I prepared twelve styles which You can use to customize this theme. To change style switch link to stylesheet from

<link rel="stylesheet" type="text/css" href="css/skin1.css" media="all" />

to style which You want e.g.

<link rel="stylesheet" type="text/css" href="css/skin12.css" media="all" />

If You want to change from default style to triangles or straight line just add link to proper css file

<link rel="stylesheet" type="text/css" href="css/triangles.css" media="all" />

If You want to setup Your own colors, go to one of th prepared skins. Rename it to what You want e.g. custom_style.css and then copy paste styles from one of skin.css files and change what You want.

If You want to change "Reservation" style to another You must edit .html files. In the header section You can find a code

<!-- #reservation starts here -->
	<div id="reservation" class="one_half last_column">
	    <a href="index.html"><img src="images/skins/reservation1.png"
		alt="Make a Reservation" width="208px" height="105px" /></a>
	</div>
<!-- end #reservation -->

Just change reservation1.png to another e.g. reservation12.png or Your own.

Customizing custom.js

Changing navigation hover effect:
$(this).find('ul:first').css({visibility: "visible",display: "none"}).show(400);

Change .show, to other effect like .fadeIn or .slideDown. You can also change the speed of effect, default is 400ms

Adding elements to opacity on hover:
$("#latest-recipes img, .follow-us li a, ...")

You can add more elements to opacity hover effect, add new elements after comma. You can also change the opacity and transition speed by changing

.fadeTo("fast", 1); and .fadeTo("fast", 0.7);

Changing number of jCarousel items on scroll
scroll: 1

Change the value '1' to another if You want to have more images than one scrolled at once

Flickr Feed
$('ul').jflickrfeed({
	limit: 9,
	qstrings: {
		id: '10729228@N07'
	},
	itemTemplate: '<li><img alt="{{title}}" src="{{image_s}}" /></li>'
});
  • 10729228@N07 - this is ID of the user on the flickr.com site
  • limit 9 - this is the number of photos which are displayed
  • itemTemplate - template of Your Flickr Feed

You can read more about options on authors site.

Twitter Feed
jQuery(function($){
	$(".tweet").tweet({
		username: "mthemes",
		count: 2,
		loading_text: "loading tweets..."
	});
});
  • username - change mthemes to Your username on twitter
  • count - number of displaying tweets
  • loading_text - text which appear before tweets load

There are more options you can use here. For more information go to the author website.

Messages

There are four default styles for message boxes: info, warning, success and error. Below You can see default structure of info message.

<div class="message-box info">
	<div class="message-inner">
		<p>Info message</p>
	</div>
</div>

To change the box message You have to change class in first div <div class="message-box info"> to one of these:

  • <div class="message-box success">
  • <div class="message-box warning">
  • <div class="message-box error">

Tabs

Default structure of the tabs:

<!-- .tabs-wrapper starts here -->
<div class="tabs-wrapper one_third">
           
    <h2>Tabs</h2>
	       
    <!-- ul.tabs starts here -->
	<ul class="tabs">
		<li><a href="#tab1">Tab 1</a></li>
		<li><a href="#tab2">Tab 2</a></li>
		<li><a href="#tab3">Tab 3</a></li>
	</ul>
    <!-- end .tabs -->
			
    <!-- .tabs-inner starts here -->
	<div class="tabs-inner">
		        
		<div id="tab1" class="tab-content">
			<!-- content goes here -->
		</div>
		            
		<div id="tab2" class="tab-content">
			<!-- content goes here -->
		</div>
                    
		<div id="tab3" class="tab-content">
			<!-- content goes here -->
		</div>
		            
	</div>
    <!-- end .tabs-inner -->
	        
</div>
<!-- end .tabs-wrapper -->

As You can see <li><a href="#tab1">Tab 1</a></li> refers to

<div id="tab1" class="tab-content">
	<!-- content goes here -->
</div>

If You want to add new tabs remember to have the same id in href and in .tab-content

Accordion

Default structure for accordion

<!-- #accordion-wrapper starts here -->
<div id="accordion-wrapper" class="one_third last_column">
           
    <h2>Accordion</h2>
           
	<!-- #accordion starts here -->
	<div id="accordion">
	           
		<div>
			<h4><a href="#">First</a></h3>
			<div><!-- content goes here --></div>
		</div>
		<div>
			<h4><a href="#">Second</a></h3>
			<div><!-- content goes here --></div>
		</div>
		<div>
			<h4><a href="#">Third</a></h3>
			<div><!-- content goes here --></div>
		</div>
	           
	</div>
	<!-- end #accordion -->
           
</div>
<!-- end #accordion-wrapper -->

H4 tag is the key here. It's the trigger for panels. It's defined in custom.js file, so if You want to change it for, let's say, h5 heading just go there and change h4 to h5

$(document).ready(function(){
	$("#accordion").accordion ({
		header: "h4"
	});
});

Portofolio (top)

Code for the filter menu

<ul class="filter"> 
	<li class="current all"><a href="#">All</a></li> 
	<li class="pizza"><a href="#">Pizza</a></li> 
	<li class="spaghetti"><a href="#">Spaghetti</a></li> 
	<li class="seafood"><a href="#">Seafood</a></li>
</ul>

Class pointing to data-type="pizza" in the list item. It's very important to add a class to Your filter when You adding another data-type. Note here that every next item should have increased id e.g. first <li data-id="id-1" data-type="pizza"> and second <li data-id="id-2" data-type="seafood">

2, 3 and 4 columns portfolios are almost the same. One thing is changing here is size of the image.

  • 430x250 for 2 columns portfolio
  • 275x180 for 3 columns portfolio
  • 200x150 for 4 columns portfolio

Code for 1 column portfolio with description on the right

<li data-id="id-7" data-type="pizza">
	<div class="fancycaption fade-zoom">
		<a href="#"><img class="fade" src="../photos/430x250/pizza4.jpg" alt="photo" /></a>
    </div>
						
	<div class="portfolio-small-desc">
                        
        <h2>Item title</h2>
                        
		<p>Your description here</p>
                            
		<a href="#">Button text</a>
    
	</div>
</li>

Code for portfolio item without description (portfolio 2, 3 and 4 column)

<li data-id="id-1" data-type="pizza" class="fancycaption fade-zoom">
	<a href="#">
		<img class="fade" src="yourimage.jpg" alt="photo" />
	</a>
</li>

Remember to change data-id and data-type!

Contact form (top)

$address = "example@themeforest.net";

Change example@themeforest.net to Your own email address.
